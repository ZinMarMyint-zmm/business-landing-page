# Business Landing Page

A modern responsive business landing page built with Next.js and Tailwind CSS.

## Features

- Responsive Navbar
- Hero Section
- Stats Section
- Features Section
- CTA Section
- Mobile Navigation

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React
